    # 资源类型

    在希沃白板的插件 API 中，如果注释说到某属性是资源并链接到本页面，那么遵循本文所述的用法。

    # Web URL

    当指定时，希沃白板会尝试去对应的 URL 下载，且不会使用任何身份认证。当下载完成后，调用方法指定的行为才会继续直到最终完成后调用回调。

    仅支持 https 和 http 协议。不支持其他协议，也不支持 http/2 协议。

    # Data URL

    如果你不希望额外去下载外部资源，那么可用 Data URL 来指定资源。
