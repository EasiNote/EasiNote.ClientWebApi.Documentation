# 插入元素

## API

```js
window.external.InsertElement("元素类型名称", "{元素数据描述的 Json 对象}")
```

### 参数说明

* 1 元素类型

| 元素类型        | 元素类型名称        | 备注 | 案例                                                                        |
|-------------|---------------------|------|---------------------------------------------------------------------------|
| Web内容呈现卡片 | WebPresentationCard |      | [《Web内容卡片元素》](https://r302.cc/4z6Vvlp?platform=enpc&channel=copylink) |

* 2 json 对象

这个与具体的元素类型相关，不同的类型，描述信息可能不一样。
